# EvoSegNet: Supplementary Material

This repository contains supplementary materials for the paper:

**EvoSegNet: EvoSegNet: Multi-Scale and Uncertainty-Aware Evolutionary Neural Architecture Search for Efficient 3D Medical Image Segmentation**

## Contents

- Training dynamics on BraTS 2021 and ACDC
- Effect of volumetric data augmentation
- Additional qualitative results (BraTS 2020 cross-dataset)
- Reproducibility details
